# PersonalBlog


个人博客网站
<p>项目结构</p>
    <p>blog---------博客内容</p>
    <p>comments-----评论内容</p>
    <p>likes--------点赞记录</p>
    <p>media--------公共资源、视频/图片</p>
    <p>PersonalBlog-django的配置文件夹</p>
    <p>reading------阅读记录</p>
    <p>static-------公共静态资源文件夹</p>
    <p>static_new---所有app的静态文件整合</p>
    <p>templates----公共html文件</p>
    <p>user---------用户 注册，登录</p>
    <p>manage.py----django启动文件</p>
    <p>requirements.txt-依赖包文件</p>
